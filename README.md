# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

**NOTA DE PRODUÇÃO:** Para que o login funcione no ambiente de produção, o domínio da aplicação (ex: `[project-id].web.app`) deve ser adicionado à lista de "Domínios autorizados" nas configurações do Firebase Authentication no Console do Firebase.
